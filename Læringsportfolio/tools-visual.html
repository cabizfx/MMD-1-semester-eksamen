Visual Studio Code - hvad er det?

Visuak Studio Code er en kodningsplatform til at skrive kode - det er et genialt værktøj til kodning da der er så meget diversitet og muligheder for lige hvad man gerne vil.

Visual Studio Code understøtter alle kodesprog - og der er mulighed for rigtig mange udvidelser til at hjælpe med kodningsoplevelsen. Man kan blandt andet for udvidelser til code-sharing, live-server, code-cleanup osv. mulighederne er mange

Hva' ka' det?

Visual Studio Code kan hjælpe med at finde fejl i ens kode - samt finde løsninger.

Hvor ka' jeg bruge det?

Det bruges til alle kodningsprojekter - jeg har f.eks brugt dem i alle vores kodningsprojekter på MMD-uddannelsen
