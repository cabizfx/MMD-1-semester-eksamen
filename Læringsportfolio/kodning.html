Kodning

Informationsteknink - hjemmesider, apps eller andre digitale løsninger spiller en kæmpe rolle i vores samfund. Derfor er det vigtigt for dem som skaber disse digitale produkter - at de kan skrive og forstå kode.

I denne sektion vil jeg fokusere på HTML og CSS - som kodesprog, da det er det eneste vi har lært på 1. semester - so far.

HTML og CSS er kodningssprog - ikke programmeringssprog. De bliver primært brugt til at skabe hjemmesider, men kan også bruges til f.eks WebApps eller UI's i spil.
